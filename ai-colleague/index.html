Triggers

兩種觸發方式

手動觸發

workflow_dispatch

透過 GitHub UI 或 gh CLI 直接觸發

# GitHub UI Actions tab → AI Colleague → Run workflow → Jira Issue Key: VSFT-4203 # gh CLI gh workflow run ai-colleague.yml \ -f jira_issue_key=VSFT-4203

適用場景：工程師自行選定任務，按需執行

外部觸發

repository_dispatch

Teams Bot / 外部系統透過 GitHub API 觸發

curl -X POST \ -H "Authorization: token $TOKEN" \ https://api.github.com/.../dispatches \ -d '{ "event_type": "ai-colleague-task", "client_payload": { "jira_issue_key": "VSFT-4203" } }'

適用場景：Teams Bot 下指令、定時排程、CI 串接
