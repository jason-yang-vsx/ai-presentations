Claude Code

Agent Teams — 官方多人協作

什麼是 Agent Teams？

  • 多個 Claude Code session 協同工作
  • 一個 Team Lead 協調分工
  • Teammates 獨立 context window
  • 彼此可 直接溝通（peer-to-peer）
  • 支援 worktree 隔離

vs Subagent 差異

特性SubagentAgent Teams
Context共用獨立
溝通單向回報雙向對話
互動無法介入可直接對話
隔離同 session各自 worktree
